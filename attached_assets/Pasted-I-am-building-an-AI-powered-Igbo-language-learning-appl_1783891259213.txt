# API Layer

The frontend should NOT contain chatbot logic.

Instead create an API service.

Example:

```typescript
POST /api/chat

{
  "message": "...",
  "conversation_id": "...",
  "user_level": "Beginner"
}
```

Expected response:

```json
{
  "response": "...",
  "lesson_state": {},
  "vocabulary": [],
  "grammar": []
}
```

The frontend should call this endpoint but for now use mocked responses if the backend is unavailable.

Create an `api.ts` service to isolate all API calls.
